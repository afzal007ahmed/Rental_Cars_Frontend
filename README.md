# Rental Cars Frontend

Frontend application for searching and booking rental vehicles.

## Overview

The application allows users to search nearby rental locations, browse available vehicles, create bookings, and update existing bookings.

---

## Features

### Vehicle Search

- Search rental locations
- Display nearby locations
- View available vehicles
- Pickup and drop location selection

### Booking

- Create booking
- Update booking
- Booking confirmation

### Authentication

- User login
- Protected pages

### Validation

- Client-side form validation using Zod
